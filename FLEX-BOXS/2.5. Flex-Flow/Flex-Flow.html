FLEX-ELEMENT-FLOW {FLEXBOX}

Flex-Flow adalah properti Flexbox yang berfungsi untuk mengatur arah susunan elemen (penulisan syntax pada parent element-nya) sekaligus menentukan apakah elemen boleh pindah baris atau tidak. Properti ini sebenarnya gabungan dari dua properti penting: flex-direction dan flex-wrap. Jadi saat kamu menulis flex-flow, kamu sedang membuat satu keputusan layout yang utuh, bukan dua pengaturan terpisah.

Ketika sebuah container memakai flex-flow, browser langsung tahu elemen disusun ke mana dan bagaimana perilakunya saat ruang tidak cukup. Misalnya, arah ke samping tapi boleh turun baris, atau ke bawah dan tetap satu kolom. Ini sangat berguna untuk layout responsif karena perilaku elemen sudah jelas sejak awal, tanpa perlu banyak aturan tambahan.

Intinya, flex-flow membuat kode CSS lebih ringkas, lebih terbaca, dan lebih terkontrol. Kamu tidak lagi mikir “arah dulu, wrap belakangan”, tapi langsung menetapkan pola layout-nya. Kalau Flexbox itu sistem penyusun, maka flex-flow adalah blueprint-nya.
